== Description ==

Allow users to choose from a set of avatars.

== Installation ==

= Manual Installation =

1. Upload the entire `/um-avatar-suggestions` directory to the `/wp-content/plugins/` directory.
2. Activate UM Avatar Suggestions through the 'Plugins' menu in WordPress.
