Contact Form 7 - Add location picker using Google Map. (Capture longitute, latitude and address using just one click on contact form 7)

== Description ==

Location Picker comes in your customers help to ease their pain caused by filling in address and exact location. 

Your customers can fill in their address and afterward pin point the exact location on the map so the delivery of the goods runs smoothly avoiding useless phone calls asking for directions.

Add location picker using Google Map. (Capture longitute, latitude and address using just one click on contact form 7)

Contact form 7 shortcode code to add map, longitute, latitude and address fields on contact form.

When user click on map or drap map marker there longitute, latitude and address fields is populated with data. 

Here is live demo: [Plugin Live Demo](http://www.wp.pluginscart.com/contact-form-7-location-picker/ "Plugin Live Demo")

== Frequently Asked Questions ==

= How to display Map? =

To display map on contact form use <b>map canvas</b> shortcode in contact form 7.

= How to display Post Code search field? =

To display post code search field on contact form use <b>postcode</b> shortcode in contact form 7.

= Can I get all populated data in email? =

Yes, you can. Just use all tags generated in mail of contact form 7.
